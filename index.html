How the Internet Works

The internet is a collection of computers that speak to each other. When one goes to a webpage, his computer sends an HTTP request to a server.The server then finds the requested HTML document and sends it back to the requestor. The requestor's browser then interprets the page and displays it on the screen. This video is a good reference for this topic.

HTML

HTML stands for Hyper Text Markup Language. The web is a collection of HTML documents. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". This video explains further.

Tags and Elements

HTML documents are made up of HTML elements. When writing HTML, we tell browsers the type of element by using HTML tags. This video does a good job of explaining.

Why Computers are Stupid

Computers are stupid because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause huge problems in a program.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an invisible box around the content inside them.
